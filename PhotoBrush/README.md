# PhotoBrush

有些图片处理的 app 有模糊背景的功能，允许你把图片的背景手动涂抹掉，目的是为了去掉和照片无关的背景内容，或是去掉一些隐私信息。

这个项目是实现用来模糊图片背景的笔刷功能。
